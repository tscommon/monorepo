# Getting started

## Installation

TypeScript common can be installed using any of the following package managers:

<Tabs>
  <TabItem value="npm" label="npm" default>
    [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) is the default package manager for [Node.js](https://nodejs.org/), and to where tscommon is published.<br />Your project is using npm if it has a `package-lock.json` file in its root folder.
    <br />
    **Run the following command in your terminal:**

    ```shell title="terminal"
    npm install @tscommon/library
    ```

  </TabItem>

  <TabItem value="yarn" label="yarn" default>
    [yarn](https://yarnpkg.com/) is a fast, reliable and secure dependency manager for [Node.js](https://nodejs.org/).
    <br />Your project is using Yarn if it has a `yarn.lock` file in its root folder.
    <br />
    **Run the following command in your terminal:**

    ```shell title="terminal"
    yarn add @tscommon/library
    ```

  </TabItem>

  <TabItem value="pnpm" label="pnpm" default>
    [pnpm](https://pnpm.io/installation) is a fast, disk space efficient package manager for [Node.js](https://nodejs.org/).
    <br/>Your project is using pnpm if it has a `pnpm-lock.yaml` file in its root folder.
    <br />
    **Run the following command in your terminal:**

    ```shell title="terminal"
    pnpm install @tscommon/library
    ```

  </TabItem>
</Tabs>
